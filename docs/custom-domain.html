Custom domain setup

GitHub Pages can publish this site at mac-photos-duplicat-finder.org, but the domain must resolve at the DNS provider before GitHub can issue HTTPS certificates.

DNS is external to this repository

The repository can include site files and workflows, but only the domain registrar or DNS provider can create the required DNS records.

mac-photos-duplicat-finder.org.  A     185.199.109.153
mac-photos-duplicat-finder.org.  A     185.199.110.153

mac-photos-duplicat-finder.org.  AAAA  2606:50c0:8002::153

Verify

dig +short mac-photos-duplicat-finder.org A
dig +short www.mac-photos-duplicat-finder.org CNAME

After DNS resolves, return to GitHub repository Settings > Pages and wait for the DNS check and HTTPS certificate provisioning to complete.

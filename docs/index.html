Capabilities

Inventory Photos libraries

Check package readability, original media counts, library size, and privacy-access failures before doing duplicate work.

Rank exact duplicates

Find byte-identical media by SHA-256, then prefer the keep copy with stronger Photos pixel and metadata signals.

Create review albums

Create a Photos album such as Lower-Quality Duplicate Candidates, then review before deleting anything.

Safety model

Shared as is

This project is provided as is, with no warranty. Photos libraries are database-backed packages. Back up your library, read the generated reports, and make final delete decisions yourself inside Photos.

No direct database writes

The scripts read Photos.sqlite only in immutable read-only mode when IDs or quality metadata are needed.

No package surgery

Files are not moved inside the .photoslibrary package. Direct package edits can corrupt Photos and iCloud state.

Albums are review aids

Removing from an album is not deleting. Deleting from the library can affect iCloud Photos and synced devices.

Core commands

These examples show the quality-ranked workflow: detect exact duplicates, rank the keep copy, then create a Photos review album.

Find exact duplicates

python3 scripts/find_exact_duplicates.py \
  --library ~/Pictures/"Photos Library.photoslibrary" \
  --output-dir reports/duplicates

Prefer the best copy by pixels and metadata

python3 scripts/rerank_exact_duplicates_by_quality.py \
  --library ~/Pictures/"Photos Library.photoslibrary" \
  --duplicate-groups reports/duplicates/duplicate_groups.json \
  --output-dir reports/duplicates-low-quality

Create a Photos review album

python3 scripts/create_photos_review_album.py \
  --library ~/Pictures/"Photos Library.photoslibrary" \
  --proposal-csv reports/duplicates-low-quality/duplicate_proposal.csv \
  --album-name "Lower-Quality Duplicate Candidates" \
  --execute \
  --batch-size 100 \
  --batch-timeout 120

Large albums are processed in batches. A batch size of 100 is conservative; smaller values are slower, larger values may make Photos stay busy longer.

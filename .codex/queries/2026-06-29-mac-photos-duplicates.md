# Consulta original

vull que generis un repo de github dins la carpeta github/personal. Es tracta de desenvolupar una app en python que permeti detectar duplicats de fotos al fotos de Mac. El programa constarà d'una sèrie d'scripts en python. El primer serà explorar les llibreries que hi ha a l'ordinador i llistar-les amb les seves característiques (accés, mida, nombre de fotos i videos...). El segon serà identificar les imatges i videos duplicats exactament i fer una proposta de moure aquests duplicats a la carpeta duplicats del fotos mac. Tot això amb la màxima seguretat per no fer malbé la llibreria existent. Si hi ha qualsevol perill, atura't i explica'm com solucionar-lo
